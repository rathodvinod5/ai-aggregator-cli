# Multi-LLM Aggregator CLI 🤖

A lightweight Node.js Command Line Interface (CLI) application that sends a single user prompt to three major AI models simultaneously: **OpenAI (GPT-4o-mini)**, **Anthropic (Claude 3.5 Haiku)**, and **Google (Gemini 2.5 Flash)**. After gathering the individual responses, it routes them to a master evaluator model (**GPT-4o**) to validate accuracy, cross-reference data points, and synthesize a single, optimized final response.

---

## 🚀 Features

-   **Parallel Execution:** Requests are fired concurrently using `Promise.all` to minimize processing latency.
-   **Local Console Execution:** No heavy frameworks (like Express) required—runs natively with the standard `node` runtime.
-   **Smart Synthesis:** Uses a high-tier evaluator model to cross-examine and summarize answers.

## 🛠️ Installation & Setup Guide

Follow these steps to clone the repository, install its necessary modules, and initialize your workspace.

### 1. Clone the Repository
Open your terminal and clone this repository down to your local machine:

```bash
git clone https://github.com
cd ai-aggregator-cli
```

### 2. Install Project Dependencies
Install the required official AI SDK components and the environment loader package:

```bash
npm install
```

### 3. Configure Your Secret Environment Keys
You must supply **your own private API keys** for OpenAI, Anthropic, and Google Gen AI for this application to operate.

1. In the root directory of the project, create a brand new file named `.env`:
   ```bash
   touch .env
   ```
2. Open the `.env` file in your preferred text editor and add the following configuration layout, substituting the placeholder text with your real credential keys:

```env
OPENAI_API_KEY="your-actual-openai-api-key-here"
ANTHROPIC_API_KEY="your-actual-claude-api-key-here"
GEMINI_API_KEY="your-actual-gemini-api-key-here"
```

> ⚠️ **Security Warning:** The `.env` file contains your private paid API credentials. It is automatically blocked via `.gitignore` and should never be committed or pushed to public code repositories.

## 💻 How to Run the Application

You can execute the script instantly by passing your custom query directly inside quotation marks as a command-line argument.

### CLI Command Syntax
```bash
node index.js "YOUR_INPUT_STRING"
```

### 📝 Example Usage

If you want to know about the current weather or standard properties of a location, pass it like this:

```bash
node index.js "What is the general climate and layout of Pune?"
```
